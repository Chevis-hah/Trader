## 当前关键架构决策 (ADR 风格)

- **CPCV 对规则策略无多路径价值**: `is_deterministic=True` 时所有 path 输出相同, PBO 退化为 tie 赌博。真正的多路径 CPCV 需要策略有 train-time 参数 (ML 模型 / 拟合阈值)。本次让报告明确标注 `det?=✓` 并把 PBO 标 `N/A`, 不再假装有统计意义。Phase 3 LightGBM aggregator 是获得真 PBO 的自然路径。
- **年化因子由调用方指定**: `summarise_cpcv_paths(bars_per_year=...)` 与 `run_cpcv_for_strategy(bars_per_year=...)`, 默认 252 (日线), 规则价格 4h = 2190, funding 8h = 1095。硬编码年化是之前的错。
- **Funding 实盘 FRAGILE 需诊断而非拍脑袋调**: `funding_harvester_backtest.py` 新增 `diagnostics` 段把总 funding_income / total_cost / ratio / per-symbol / top winners/losers 全部暴露, 下次迭代用这些数字精准定位 "成本吃光 vs 收入不足 vs 某 symbol 拖后腿"。
- **本地 agent 的 OSError 降级**: 在 WSL 缺 `libgomp.so.1` 但 `pip install lightgbm` 成功的场景下, import 会抛 OSError。`_try_import_lgbm` 除 ImportError 外捕获 OSError, 与未安装路径一致降级到 sklearn。
